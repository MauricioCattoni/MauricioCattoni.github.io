Sistema de control de horario de empleados

Repositorio en Github

El sistema funciona con un framework que se encarga que recuperar la informacion de los usuarios por medio de un front end generado en archivos php, con html y css, y funciones en Javascript. Los datos de la seccion y del usuario son manejados por variables en php El sistema se divide en acciones, vistas y controladores. Las vistas representan cada pantalla que tiene el sistema, las acciones nos permiten modificar o agregar los datos existes haciendo llamadas a la base de datos y los controladores son particulares para cada tabla de la base de datos.
